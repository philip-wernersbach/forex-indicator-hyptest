Forex-indicator-hyptest

A forex market indicator that uses hypothesis testing to make market predictions.

Download .zip Download .tar.gz View on GitHub

Run the indicator!

Input the data below and press the button to run the indicator. Sample data is already provided, but you can erase it and enter your own.

The sample data is taken from EUR/USD, and the data points are the low prices at 12:00am, leading up to 09/12/2012. The currency pair reached the exiting price on 09/14/12, so if it works it will display that the currency pair may reach the exiting price.
